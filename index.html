Shopify Product Images Hosting

This repository hosts product images for Shopify import using GitHub Pages.

Images are available at: https://alex-halloran.github.io/shopify-product-images/images/[filename]

How It Works

The automated script:

  1. Downloads images from Dropbox links
  2. Uploads them to this GitHub repository
  3. Serves them via GitHub Pages
  4. Generates a mapping between Dropbox URLs and GitHub Pages URLs

Usage

See the README.md file in the repository for detailed usage instructions.
